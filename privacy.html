5. Data Storage & Security

5.1 Storage

Your data is stored in:

5.2 Security Measures

We implement the following security controls:

  • All data transmitted over HTTPS / TLS 1.3
  • Financial inputs anonymized after report generation

  • No storage of full card numbers or payment credentials

5.3 Retention

We retain personal data for the minimum period necessary:
